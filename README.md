# Browser ZIP Extractor

Small browser-based utility for reading ZIP files and displaying their contents client-side.

## Features

- Upload a ZIP file in the browser
- Inspect extracted file names/content
- No backend required
- Static HTML/CSS/JS app

## Run locally

Open `index.html` in a browser or serve it with:

```bash
python3 -m http.server 8000
```

## Notes

Useful as a lightweight demo/tool for client-side archive inspection.
